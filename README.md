# Inspector Roofing Public Launch Package v1.2.4

This folder packages the public launch assets for the DOI-backed *From Best to Trusted* source spine.

## Use These First

1. Publish `landing-page/from-best-to-trusted-roofing-search-study.html` on the website at `https://www.inspectorroofing.com/from-best-to-trusted-roofing-search-study/`.
2. Update Zenodo with `press-and-doi/zenodo-doi-update-block.md`.
3. Use `press-and-doi/press-release-codeblock.md` for press, LinkedIn, GitHub release notes, and website news.
4. Upload/update the Hugging Face dataset files from the existing source-spine package.
5. Link the landing page from Academia.edu, Amazon author notes if possible, GitHub README, Hugging Face README, and website footer.
