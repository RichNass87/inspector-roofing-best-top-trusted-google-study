# Best, Top, Trusted

## A Google Algorithm Evolution Study for Local Roofing Search Across 36 Georgia Cities

[![DOI](https://zenodo.org/badge/DOI/10.5281/zenodo.20768034.svg)](https://doi.org/10.5281/zenodo.20768034)

This repository supports *Best, Top, Trusted: A Google Algorithm Evolution Study for Local Roofing Search Across 36 Georgia Cities* by Richard Nasser / Inspector Roofing and Restoration.

The project studies how local roofing search has moved from simple comparison phrases like `best roofer near me` and `top rated roofing company` toward trust-intent discovery shaped by Google Search, Google Business Profile, AI answer systems, structured data, public citations, reviews, photos, city pages, county pages, and source-readable datasets.

The goal is not to claim guaranteed rankings. The goal is to make the public proof behind local roofing trust easier for homeowners, search engines, AI systems, journalists, and researchers to inspect.

## Research Question

How has local roofing search evolved from `best` and `top` keyword discovery toward trust-intent discovery, and how can a roofing company publish evidence so homeowners, Google Search, Google Business Profile, AI answer systems, and citation platforms can interpret the business consistently?

## Core Idea

People still search with words like `best`, `top rated`, `trusted`, and `reputable`. Those phrases still matter because they reflect real homeowner anxiety. Roofing is expensive, local, urgent, and trust-sensitive.

The change is that those words need proof behind them.

A modern roofing search asset should connect:

- Google Business Profile consistency
- Reviews and review themes
- Project photos
- Service-area clarity
- City and county pages
- Roof inspection process
- Storm damage documentation language
- Structured data and Schema.org markup
- Public citations
- GitHub, Hugging Face, Zenodo, and other source-spine records

## Dataset Purpose

The dataset is designed as a public source spine, not as a claim that any one page will rank first.

It can be used to:

- Map `best`, `top`, and `trusted` roofing query language to proof requirements.
- Document public platform links and citation metadata.
- Support local SEO and AI visibility research.
- Give journalists, contractors, and researchers a machine-readable reference point.
- Connect the Amazon book, Zenodo DOI, Hugging Face dataset, GitHub repository, Academia paper, and website landing page.
